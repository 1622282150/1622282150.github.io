# VSCode 常见问题

# 001.完美解决： VSCode 中文显示

  • 第一步：找到 扩展，搜索 chinese 或者 访问在新窗口打开，点击 Install 进行下载，然后重启 VSCode；

  • 第二步：Ctrl+Shift+P 或者  ⌘ +Shift+P，打开命令行，搜索 language，选择 Configure Display Language，再选择 zh-cn 即可。

    参考资料：完美解决：VSCode 中文显示在新窗口打开

# 002.VSCode 中保存后会在当前目录下自动生成 dist 目录

产生原因：

Sass/Less/Stylus/Pug/Jade/Typescript/Javascript Compile Hero Pro 中文介绍在新窗口打开

解决办法：

关闭 compile-hero 插件

  • 在设置中搜索 compile-hero 插件
  • 关闭所有自动生成 dist 目录的选项

# 003.VSCode 无法更改文件夹名称

解决办法：

删除文件内node_modules目录，再进行重命名操作。
